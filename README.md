# [圖文留言板](https://board.taipeilife.info/)

本專案提供使用者圖文留言功能

## Demo
點擊網址前往圖文留言板網頁：https://board.taipeilife.info/

## 使用技術
* Python Flask
* 網頁建構於AWS EC2
* 於AWS RDS建立資料庫
* 運用AWS S3儲存圖片
* 透過AWS CloudFront建立CDN系統
* 申請SSL憑證實踐HTTPS
